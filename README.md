# Todo App — Priyanshu Singh

A clean and premium todo application built with vanilla HTML, CSS, and JavaScript.

## Live Demo

[View Live App](https://priyanshu123-coder.github.io/todo-app/)

## Features

- Add, complete, and delete tasks
- Priority levels — High, Medium, Low
- Filter tasks — All, Pending, Done
- Progress bar with percentage
- Data persists using localStorage
- Fully responsive design

## Tech Stack

| Technology | Usage |
|------------|-------|
| HTML5 | Structure |
| CSS3 | Styling and layout |
| JavaScript ES6 | Logic and DOM manipulation |
| localStorage | Data persistence |

## Getting Started

```bash
git clone https://github.com/Priyanshu123-coder/todo-app.git
```

Open `index.html` in browser or use Live Server.
